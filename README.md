# Agar.io Clone

A browser game inspired by Agar.io built with HTML5 Canvas and JavaScript.

Play here:
https://ahaangupta720-blip.github.io/agar.io/

## Gameplay

You control a cell and move with your mouse.  
Eat small food beads to grow and survive against AI bots.

If a bot is larger than you, it can eat you.

The goal is to reach the size goal and level up.

## Features

- Large scrolling world
- Camera follows player
- AI bots
- Leaderboard with size counter
- Minimap
- Infinite difficulty levels
- Level-up screen
- Save / load system using a save string
- Difficulty scaling

## Level System

Each level requires reaching **700 size**.

After reaching the goal:

- The level increases
- Your size resets
- Growth becomes harder

Food required for +1 size doubles every level.

Example:

Level 1 → 1 food per size  
Level 2 → 2 food per size  
Level 3 → 4 food per size  
Level 4 → 8 food per size  

This continues infinitely.

## Controls

Move Mouse — Move cell

Save Button — Generate save string

Load Button — Load save string
